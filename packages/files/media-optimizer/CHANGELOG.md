# Changelog

All notable changes to `@contfu/media-optimizer` are documented here.

## [0.0.9] - 2026-04-18

### Added

- AVIF input support: both buffer and stream inputs are now accepted
- 10-bit AVIF inputs are passed through unchanged (no-op passthrough)

### Changed

- Bumped `@m4k/common` to 0.2.7 and `m4k` to 0.3.3

## [0.0.8] - 2026-04-18

### Added

- Named variant preset support aligned with `@contfu/contfu` media API

## [0.0.7] - 2026-04-16

### Changed

- Updated `m4k` dependencies

## [0.0.6] - 2026-04-14

Initial public release.
